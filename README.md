# The_Data_Story_Dashboard
End-to-end data pipeline for my SAM AI Technologies internship. Includes a Python web scraper for global company data, automated data cleaning with Pandas, and an Exploratory Data Analysis (EDA) dashboard. Features professional Seaborn visualizations, including revenue distribution and business efficiency insights.
Revenue Analysis & Data Pipeline
This repository documents my progress and technical deliverables during my Data Analytics Internship at SAM AI Technologies. The project transitions from raw data extraction to actionable business intelligence.

🛠️ Project Phases

Task 1: Automated Data Sourcing: Developed a Python-based web scraper using BeautifulSoup to extract real-time financial data from Wikipedia.

Task 2: Data Engineering: Performed advanced cleaning using Pandas, including header normalization, null value handling, and data type transformation (converting objects to float64/int64).

Task 3: Visual Storytelling: Built a comprehensive analysis dashboard using Seaborn and Matplotlib.

Bar Charts: Top 10 Global Leaders by Revenue.

Pie Charts: Industry-wide market concentration.

Scatter Plots: Revenue vs. Profit efficiency analysis.

💡 Key Insights

Identified Retail and Energy as the dominant sectors in the top 50 global companies.

Analyzed the profitability gap between high-revenue retail giants and high-margin technology firms.
